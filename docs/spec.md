# Project Specification

## Project Name
Copilot Task API

## Purpose
A simple REST API for managing tasks (CRUD operations).

## Features
- Create a task
- Get all tasks
- Get task by ID
- Update task
- Delete task

## Tech Stack
- Node.js
- Express
- Jest for testing

## API Contract

### POST /tasks
Create new task

### GET /tasks
Return all tasks

### GET /tasks/:id
Return task by ID

### PUT /tasks/:id
Update task

### DELETE /tasks/:id
Delete task
